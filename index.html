Heuristic Review

Evaluation of thecannonsmouth.com

  • Home
  • Mental Model
  • User Constraints

Problem:

I decided to do a review on this site. As I was reviewing the site, I found a few potential usability issues that I would like to address the owner. I used the following principles to run my evaluation.

  • Match Mental Model
  • Account for User & Environmental Constarits

Process:

My process was to go through the site, identify areas of attention using the 2 of the heuristics that are mentioned above. By concentrating on those, I was able to find some problem areas and put together a small prototype of my changes that would, in my opinion, improve the user experience of this design.

Decisions:

Some of the decisions that I made were based on methods that I have learned in various classes that I have taken at UVU.

The reasonings behind why I did what I did are on the appropriate pages for the two heuristics that I ran.

Some aspects of this design did not really meet up to what I think should be a focus in UX.

Outcome:

The outcome? Well, I haven't completely finished the complete analysis. I still have a few things here and there to take care of.

My next steps are to go back and revise a few key points in this design that I missed. I will probably meet with the owner of this site and see if there's anything that he would like to see done. I will try to run a full usability test on the website if permitted.

Reflection:

What went well? Definitely the prototypes that I put together. I have a few that I would like to show the owner and see if he would implment them.

What didn't go well? I had originally thought that a website such as this would have a few more usability problems but, I was wrong. I think that the next time I do something like this, I would find a bigger project to take on.

The biggest failure to this whole project, for me, was the simple fact that some of the software I was trying to use to build the documentation itself did not want to cooperate with the design that I was trying to implement
