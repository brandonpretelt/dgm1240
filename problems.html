Overview

Here is an overview of the problems identified.

  1. The site is fairly easy and standard to use. Aside from a few things, it's your standard e-commerce site.
  2. There is no account feature. I think that would help this site progress.
  3. In order to have better consistency with other systems, I think that the menu would be better positioned up on the top right or something of the sort. On mobile, a hamburger menu would be ideal.
  4. I like that search is where it is expected.

In Depth

Search is in a good place but, I think it would be more beneficial to get rid of the line that is separating the top menu from the rest of the content.

That side menu, while being a decent place for it, I think would be better up top to better other systems. It can help keep with consistency that a majority of people expect from using a website.

In this design, I also have a bit of a problem with where the menu is placed on mobile.

In my tests, I found that it was a bit more cumbersome to have it placed in the middle of the page like and it takes up some real estate.

I propose to change it to the below. The menu right up next to the cart. This has more of a consistent feel and helps match up with other more familiar systems.

In my prototype, the menu bar does not have that horizontal rule at all

Something else I have noticed in my evalution is that there is no "my account" feature. I feel like this website would benefit from it because the user would have a place to keep everything stored.
